# PMP Control

Plataforma web modular para la gestión, planificación y trazabilidad del Programa de Mantenimiento Preventivo (PMP) en entornos hospitalarios.

---

## 🏥 Contexto

Este proyecto fue desarrollado en el marco del trabajo de título de Ingeniería en Computación e Informática (2025), enfocado en resolver problemáticas reales del Hospital Dipreca, relacionadas con:

- Falta de trazabilidad en mantenimientos
- Gestión manual en herramientas como Excel
- Dificultad en el seguimiento y control de actividades
- Cumplimiento de normativas ministeriales

El sistema considera lineamientos de:

- Resolución Exenta N°1341
- Norma Técnica N°0226

---

## 🎯 Objetivo

Optimizar la gestión del mantenimiento preventivo mediante el desarrollo de una plataforma web que:

- Centraliza la planificación y ejecución
- Mejora la trazabilidad de los procesos
- Permite control y seguimiento en tiempo real
- Facilita el cumplimiento normativo

---

## ⚙️ Arquitectura y Tecnologías

**Arquitectura:**
- Cliente – Servidor
- Estructura en capas: Presentación, Negocio y Datos

**Stack Tecnológico:**
- Frontend: React + Vite
- Backend: Node.js + Express
- Base de datos: PostgreSQL

---

## 🚀 Funcionalidades principales

- Gestión de usuarios con roles diferenciados
- Registro de equipos e instalaciones
- Planificación y calendarización automática
- Registro de mantenimientos con evidencia
- Generación de reportes
- Auditoría y trazabilidad de acciones

---

## 🚧 Estado del proyecto
